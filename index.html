    JavaScript TicTacToe

    This code is an HTML document that creates a webpage for playing the game of Tic Tac Toe. The webpage contains a board consisting of nine squares arranged in a 3x3 grid. Each square is represented by a div element with the class name "square". The link tag in the head section of the HTML document references a CSS file called "style.css" which is used to style the webpage. The script tag at the end of the body section references a JavaScript file called "script.js" which is used to provide the game logic and handle user interactions.

    Savings Calculator

    This is an HTML page that contains a savings calculator. It includes a brief introduction to the importance of savings, followed by a form with fields for the user to input their current savings amount, savings goal, and monthly deposit amount. When the user clicks the "Calculate" button, a JavaScript function is executed that calculates the number of months needed to reach the savings goal based on the user's inputs. The result is displayed below the form in a paragraph element with the id "result".
